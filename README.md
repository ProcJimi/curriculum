# SASCourse
SAS Example Programs: These programs are written for Stat 4197/6197 students to reinforce the concepts of SAS programmning language, SQL, macro facility, and SAS/IML.
They are either written by the course instructor or adapted from various sources including SAS Documentation, SAS Blogs, and SAS-L. Please let the instructor know of any errors if you have found. Don't circulate the material.
